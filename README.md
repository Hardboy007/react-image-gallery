# React Image Gallery

A modern image gallery app built with React that fetches images from an API and displays them with smooth UI and optimized performance.

## 🚀 Features
- API-based image fetching (Picsum API)
- Pagination (Next / Prev)
- Lazy loading for images
- Skeleton loading UI
- Search filter by author
- Responsive design with Tailwind CSS

## 🛠 Tech Stack
- React (Hooks)
- Axios
- Tailwind CSS

## ⚡ Performance Optimizations
- Lazy loading images
- Optimized image sizes
- Smooth loading transitions

```bash
git clone https://github.com/Hardboy007/react-image-gallery.git
cd react-image-gallery
